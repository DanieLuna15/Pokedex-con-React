1. Instalar Node JS, GIT, y VsCode
2. Ir a la pagina https://create-react-app.dev/docs/adding-typescript/
3. Crear carpeta del proyecto y abrir terminal dentro de este
4. Ejecutar npx create-react-app my-app --template typescript para cambiar "my-app" por el nombre de nuestro proyecto
5. Una vez creada la carpeta ejecutar npm run start
